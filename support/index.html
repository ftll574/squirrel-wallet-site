常見問題

AI 辨識失敗怎麼辦？

App 內建三家視覺語言模型（OpenRouter / Gemini / Groq）會自動 fallback，但若三家都暫時無法回應，可能是當天網路或 API 服務問題。建議：

  • 稍後重試（5–10 分鐘後通常恢復）
  • 確認手機網路正常
  • 實在不行可以手動輸入票券資訊或改掃條碼
票券資料會上傳到雲端嗎？

不會。所有票券、優惠券、條碼、截圖都儲存在你裝置的本地資料庫（Hive）中，不會上傳到任何伺服器。詳情請看 隱私權政策。

唯一會「離開裝置」的資料是 AI 辨識時主動傳送的圖片 — 而且不附帶任何使用者識別資訊。

換新手機後，票券資料會跟著過去嗎？

目前需要手動操作：在舊手機進入「設定 → 備份與還原」匯出備份檔案（`.zip`），轉到新手機後再從同一個入口匯入即可。

未來版本會加入 iCloud 自動同步功能，使用 Apple ID 跨裝置自動同步。

eSIM 為什麼點下去沒反應？

eSIM 安裝需要：

  • iOS 12.1+ 或 Android 9+ 且裝置支援 eSIM（iPhone XS 之後的機型大多支援）
  • 正確的 LPA 啟用碼（格式為 LPA:1$smdp.address$activationcode）

若點下去 iOS 沒跳出系統設定，請確認：(1) 裝置有支援 eSIM、(2) LPA 字串格式正確（可在票券詳情頁手動編輯）。

條碼掃描掃不到？

請確認：

  • 已授權相機權限（設定 → Squirrel Wallet → 相機 開啟）
  • 光線充足、條碼平整無皺摺
  • 手機與條碼距離約 10–20 公分

實在掃不到可以截圖票券條碼，從相簿選取後用 AI 辨識備援。

怎麼回報 bug 或建議新功能？

請寫信到 support@squirrelwallet.app，附上：

  • 你用的裝置型號 + iOS 版本
  • App 版本（在「設定 → 關於」看得到）
  • 問題重現步驟（如果方便的話附上截圖）
App 是免費的嗎？

是的，目前完全免費，沒有廣告、沒有訂閱、沒有 in-app purchase。AI 辨識使用免費 VLM 模型額度。

FAQ

AI recognition failed — what should I do?

The App has three Vision Language Model providers (OpenRouter / Gemini / Groq) and falls back across them automatically. If all three are temporarily down, you can:

  • Try again in 5–10 minutes
  • Check that your network connection is OK
  • Or just enter the ticket details manually / scan the barcode
Is my ticket data uploaded to the cloud?

No. All tickets, coupons, barcodes, and screenshots are stored locally on your device using Hive. Nothing is uploaded to any server. See the Privacy Policy for full details.

The only data that ever leaves your device is the image you explicitly send for AI recognition — and even then, no user-identifying metadata is attached.

Will my data move with me to a new phone?

Right now this is a manual step: open Settings → Backup & Restore on the old phone, export the backup file (`.zip`), transfer it to the new phone, and import from the same screen.

A future version will add automatic iCloud sync (using your Apple ID) so this happens transparently.

The eSIM install button does nothing?

eSIM installation requires:

  • iOS 12.1+ or Android 9+, on hardware that supports eSIM (most iPhones since iPhone XS)
  • A valid LPA activation code in the form LPA:1$smdp.address$activationcode

If tapping the button doesn't bring up the system installer, check that (1) your device supports eSIM, (2) the LPA string is well-formed (you can edit it on the ticket detail screen).

Barcode scanning isn't picking anything up?

Please check:

  • Camera permission is granted (Settings → Squirrel Wallet → Camera)
  • The barcode is well-lit and not too creased
  • You're holding the phone roughly 10–20 cm from the barcode

If scanning still fails, you can screenshot the ticket and use the AI recognition path from your photo library as a fallback.

How do I report a bug or request a feature?

Email support@squirrelwallet.app with:

  • Your device model + iOS / Android version
  • The App version (visible in Settings → About)
  • Steps to reproduce, plus a screenshot if possible
Is the App free?

Yes — completely free, no ads, no subscription, no in-app purchases. The AI recognition uses the free tier of each VLM provider.
