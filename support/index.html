繁體中文

Squirrel Wallet 是獨立開發的小工具，目前由一位工程師維護。 我們盡量在 48 小時內回覆問題。

常見問題

AI 辨識失敗 / 結果不準怎麼辦？

我們同時對接 OpenRouter、Google Gemini、Groq 三家免費 VLM，會自動 fallback。如果三家都失敗，可能是：

  • 網路問題 — 切換到行動網路 / Wi-Fi 再試一次。
  • 圖片內容不是票券 — AI 找不到票券特徵會回傳「無法辨識」，請用「手動新增」。
  • 當日免費額度用完 — Free tier 每月 5 次，Pro 每月 300 次（軟上限）。

辨識結果不滿意時，可以在 Step 3 預覽畫面直接編輯任一欄位再儲存。

條碼掃描掃不到 / 太小看不清楚？

支援 13 種條碼格式（含 EAN-13、Code 128、QR Code 等）。掃描畫面點擊 手電筒可開閃光輔助。如果票券條碼太小，可以改用「拍照新增」走 AI 路徑，AI 會自動把條碼放到票券卡上。

結帳時顯示條碼太小？打開該票券詳情頁，點條碼會跳出全螢幕亮屏模式， 手機螢幕會自動調到最亮以利掃描槍讀取。

eSIM 安裝沒反應？

eSIM 安裝是透過系統 URL 喚起 iOS / Android 內建的 eSIM 設定流程。 若沒反應請確認：

  • 裝置支援 eSIM — iPhone XS / SE 第 2 代以後、Pixel 4 以後的多數 Android。
  • LPA 字串完整 — 拍照辨識可能漏字，請對照電信商 email 確認 LPA URL 完整。
  • 該電信商支援自動安裝 — 部分電信商需要 QR Code 手動掃，不支援 LPA URL 自動安裝。
備份 / 還原怎麼做？資料會傳到雲端嗎？

「設定 → 備份」會在裝置本機產生一個 .squirrel 加密 ZIP， 然後跳出系統分享面板由您決定送到哪裡（iCloud、Google Drive、 AirDrop、Email、訊息 …）。我們沒有自己的雲端伺服器， 備份檔不會經過我們的後端。

還原時兩個模式：「覆寫」會清空本機資料用備份內容取代；「附加」會 合併到現有資料上，分類名稱衝突時讓您選怎麼對映。

Pro 訂閱怎麼取消 / 退款？

訂閱由 Apple App Store / Google Play 管理，不經過我們：

  • iOS：設定 → Apple ID → 訂閱項目 → Squirrel Wallet → 取消訂閱
  • Android：Google Play → 個人資料 → 付款與訂閱 → 訂閱 → Squirrel Wallet → 取消

取消後仍可使用 Pro 到目前計費週期結束。退款請走平台流程 (Apple Report a Problem / Google Play Help)。

換手機 / 重灌 App，資料會不見嗎？

會。因為我們不收集帳號 / 不上雲端，所有資料只存在裝置本機。 建議定期透過「設定 → 備份」匯出，把 .squirrel 檔案 存在 iCloud Drive / Google Drive。換機時開 App → 設定 → 還原 → 選備份檔即可。

怎麼刪除所有資料？

「設定 → 清除所有資料」紅色按鈕。為避免誤觸，需要您手動輸入 「刪除」二字才會啟用確認鈕。詳細清除 / 保留項目見 隱私權政策第 5.3 節。

我可以匯入其他票券 App 的資料嗎？

目前不行 —— 因為其他 App 多半沒提供標準匯出格式。如果你的舊資料 有 JSON / CSV 結構，可以來信 support@squirrelwallet.app 提需求， 看能不能加 importer。

FAQ

AI recognition failed / result inaccurate — what now?

We route through OpenRouter, Google Gemini, and Groq free-tier VLMs with automatic fallback. If all three fail, the cause is usually one of:

  • Network — switch between Wi-Fi and cellular and retry.
  • Image isn't a ticket — AI returns "unrecognised" when no ticket features are detected; use Manual Entry.
  • Quota exhausted — Free tier is 5 scans / month, Pro is 300 / month (soft cap).

If the AI got fields wrong, every field on the Step 3 preview is editable before save.

Barcode scanner won't read / barcode too small at checkout?

13 barcode formats supported (EAN-13, Code 128, QR, etc.). Tap the torch button in the scanner for low-light help. If a ticket's barcode is too small, use "Add via photo" — AI extracts the barcode and renders it large on the ticket card.

At checkout, tap the barcode on a ticket's detail screen to enter full-screen mode; the phone brightness ramps to max automatically so checkout scanners read cleanly.

eSIM install does nothing when I tap?

eSIM install fires the system URL handler for iOS / Android's built-in eSIM setup. Common causes when nothing happens:

  • Device doesn't support eSIM — iPhone XS / SE 2nd gen or later; most Pixel 4+ and recent flagship Androids.
  • LPA string truncated — photo recognition can miss characters; verify the LPA URL against your carrier email.
  • Carrier requires manual scan — some carriers only accept QR-code scans, not LPA-URL auto-install.
Backup / restore — does my data go to your cloud?

Settings → Backup writes an encrypted .squirrel ZIP on the device, then opens the OS share sheet so you choose where to send it (iCloud, Google Drive, AirDrop, email, messages …). We do not operate any cloud servers — backups never touch our backend.

Restore offers two modes: "Overwrite" wipes local data and replaces it; "Append" merges into existing data with a category- mapping sheet when names collide.

How do I cancel / refund Pro?

Subscriptions are managed by Apple App Store / Google Play, not us:

  • iOS: Settings → Apple ID → Subscriptions → Squirrel Wallet → Cancel
  • Android: Google Play → Profile → Payments & subscriptions → Subscriptions → Squirrel Wallet → Cancel

You retain Pro until the current billing period ends. Refunds go through the platform: Apple Report a Problem or Google Play Help.

Will my data survive a phone change or reinstall?

Not automatically — because we don't collect accounts or sync to the cloud, all data lives on-device. Export regularly via Settings → Backup and keep the .squirrel file in iCloud Drive / Google Drive. On a new device, open the app → Settings → Restore → pick the backup.

How do I delete everything?

Settings → Reset all data (red button). To prevent accidental taps, you must type "刪除" / "DELETE" before the confirm button activates. See Privacy §5 for the exact wipe / preserve allowlist.

Can I import data from another ticket app?

Not yet — most ticket apps don't expose a standard export. If your old data is in JSON / CSV, email support@squirrelwallet.app with the format details and we'll see if an importer is feasible.
